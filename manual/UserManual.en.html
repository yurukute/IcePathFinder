Import maze from file

Step 2: Go to file’s location and choose the file, then click Open

Result: Maze loaded

Solve the maze

Step 2: Choose the algorithm

The result: Ice path found.

Change ice path’s color

Step 1: Select Change path’s color  in Option menu

Step 2: Select the path you want to change color

Step 3: Select any color then click OK

Step 4: Click OK again to confirm

Step 5: Repeate the solving steps to resolve the maze with the new path’s color

Result: Path’s color changed

Change interface language

Step 1: On Option menu, choose Change language, then click on the language to prefer

Result: Language changed
